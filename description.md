# Автоматический перевод текста в верхний регистр

| Параметр            | Значение          |
| ------------------- | ----------------- |
| Ограничение времени | 2 секунды         |
| Ограничение памяти  | 64Mb              |
| Ввод                | Стандартный ввод  |
| Вывод               | Стандартный вывод |

## Задача

Вы работаете над системой обработки текстов, которая должна автоматически приводить все важные сообщения в верхний регистр. Это поможет лучше организовать визуализацию информации и выделять ключевые сообщения в интерфейсе.

Для этой цели вы решили использовать декоратор, который автоматически будет преобразовывать результат функции в верхний регистр (Подразумевается, что функции возвращают только строки). Ваша задача — написать такой декоратор `uppercase_result` и применить его к функции, возвращающей текстовое сообщение.

Программа должна корректно выводить текст в верхнем регистре.

## Формат декоратора

Декоратор должен проверять результат выполнения функции:
- Если результат — строка, преобразовать её в верхний регистр.
- Если результат не является строкой, вернуть результат без изменений.

## Формат вывода

Программа должна вывести результат вызова функции:
- Если функция возвращает строку, вывод должен быть в верхнем регистре.
- Если функция возвращает не строку, результат должен быть выведен без изменений.
